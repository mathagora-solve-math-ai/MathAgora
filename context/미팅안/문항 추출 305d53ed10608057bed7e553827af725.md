# 문항 추출

### **1. 영어권 수능 SAT에서의 문항별 분리**

- 기존의 [OCR 모델 + Rule-based] 방법 결과
    - 문항을 아예 식벽하지 못함
    - 알고리즘을 수정하면 95%이상 성능으로 문항을 분리할 수 있을것으로 보임
- 기존의 수능문제지로 학습한 모델 결과
    - 약 80% 성능으로 문항을 crop
    
    → 추가 데이터 학습
    

|  | 수능시험지(300장) | 영어권 SAT 시험지(20장) |
| --- | --- | --- |
| OCR model + Rule based | 298/300 (99%) | 0/20 (0%) |
| Trained Detector | 256/300 (85%) | 14/20 (70%) |

**Ground Truth (GT)**

<aside>

첫째항과 공비가 모두 양수 $k$인 등비수열 $\{a_n\}$이 $$\frac{a_4}{a_2} + \frac{a_2}{a_1} = 30$$을 만족시킬 때, $k$의 값은? [3점]
① 1 ② 2 ③ 3 ④ 4 ⑤ 5

</aside>

***paddle_ocr_vl***

```
3. 첫째항과 공비가 모두 양수 k인 등비수열 {a_n} 이\frac{a_4}{a_2} +\frac{a_2}{a_1} = 30 을 만족시킬 때,
k의 값은? [3점]
① 1 ② 2 ③ 3 ④ 4 ⑤ 5
```

- **문장 인식 정확**
- **수식 구조 유지**
- LaTeX 블록 구분 일부 누락
- Latex 문법구조 불일치
