# 연구 요약

| **기존 연구** | **수행 범위** | **Cross-Solution** | **의미 Grouping** | **Alignment** |
| --- | --- | --- | --- | --- |
| PRM800K (ICLR 2024) | 단일 풀이 step 정오 판별 | ✗ | ✗ | ✗ |
| ROSCOE (ICLR 2023) | Step↔Source alignment | ✗ | ✗ | △ (단방향) |
| Math-Shepherd (ACL 2024) | MCTS 기반 step scoring | △ (트리 내) | ✗ | ✗ |
| HRM (2025) | 연속 step 병합 평가 | ✗ | △ (물리적) | ✗ |
| SCoT (2025) | 다중 모델 독립 평가 | △ (결과만) | ✗ | ✗ |
| **본 연구 (Ours)** | 다중 풀이 step 정렬+그룹핑 | **✓** | **✓** | **✓** |
- PRM800K (ICLR 2024)
    - 단일 풀이 내에서 각 step이 맞는지 틀리는지를 인간이 annotation한 데이터셋이다. 80만 개의 step-level label을 제공하지만,
    - 이건 어디까지나 하나의 풀이 안에서 "이 step이 correct한가"를 판별하는 것이지, 서로 다른 풀이의 step을 비교하거나 의미적으로 묶는 작업은 전혀 수행하지 않는다.
- ROSCOE (ICLR 2023)
    - 그나마 alignment이라는 개념을 도입한 연구인데, 여기서의 alignment은 reasoning step을 source 문장(문제 원문)에 대응시키는 단방향 alignment이다.
    - 즉 "이 step이 문제의 어떤 조건을 참조하고 있는가"를 보는 것이지, "모델 A의 step과 모델 B의 step이 같은 조작인가"라는 cross-solution alignment은 아니다.
- Math-Shepherd (ACL 2024)
    - MCTS를 활용해서 각 step의 quality를 자동으로 scoring하는 연구다. 같은 MCTS 트리 안에서 여러 경로가 탐색되므로 제한적인 cross-path 비교가 가능하긴 하지만,
    - 이건 동일 모델 내 탐색 경로 간 비교일 뿐이고, step 간 의미적 grouping이나 alignment은 수행하지 않는다.
- HRM (2025)
    - 기존 PRM을 확장해서 연속된 2개 step을 물리적으로 병합하여 평가하는 계층적 접근을 제안했다. "그룹핑"에 가장 가까운 시도이긴 한데,
    - 이건 단순히 인접한 step을 묶는 것이지, "의미적으로 같은 수학적 조작을 수행하는 step"끼리 묶는 것이 아니다.
        - 예를 들어 step 3과 step 7이 둘 다 "양변 정리"를 하더라도, 인접하지 않으면 같은 그룹으로 묶이지 않는다.
- SCoT (2025)
    - GPT-4o, DeepSeek-V3, Gemini-2.0 세 모델의 풀이를 비교한 연구로, 유일하게 cross-solution 비교를 시도한다.
    - 하지만 여기서의 비교는 각 모델 풀이를 5개 차원(정답, step 완전성, step 타당성, 계산 정확도, 문제 이해도)으로 독립적으로 평가한 뒤 점수만 나란히 놓는 방식이다. "모델 A의 어떤 step과 모델 B의 어떤 step이 대응된다"는 step-level alignment은 없다.

# 향후 방향

1. 문제점 정의
    - 기존 방법론은
        - 단일 풀이 정답 여부 (ORM)
        - 개별 step의 정답 여부 (PRM) 에 초점
    - 그래서 문제는,
        - 모델 간 풀이 전략 비교 불가
        - step 단위의 구조적 대응 관계 부재
2. 우리 연구의 가치
    - 분석적 가치
        - 모델 간 reasoning strategy의 차이를 flow map 이용해 구조적으로 시각화
    - 평가적 가치
        - step 단위 정답 여부를 넘어서, 세분화된 모델별 능력 profiling이 가능
    - 교육적 가치
        - 같은 문제에 대한 다양한 풀이 경로 구조화
        - 학습자에게 다양한 접근법을 제시하는 체계적인 교육 도구로 확장 가능
