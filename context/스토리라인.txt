LLM과 VLM의 발전으로 수학 문제를 자동으로 풀이하는 기술은 빠르게 확산되고 있다. 특히 최근에는 이미지 기반 입력을 직접 처리하거나, OCR을 통해 텍스트를 추출한 뒤 풀이를 수행하는 서비스들이 등장하면서, AI 기반 수학 풀이가 대중화되고 있다.

그러나 대부분의 기존 시스템은 단일 문항 입력(single-problem input)을 전제로 한다. 사용자는 문제를 하나씩 촬영하거나 직접 크롭해야 하며, 텍스트 기반 입력의 경우에는 수식을 다시 입력해야 하는 번거로움이 따른다. 이러한 방식은 시험지(PDF 한 장)나 문제집 페이지처럼 **다수 문항이 포함된 실제 환경과는 괴리**가 있으며, 대량 문제 처리나 체계적인 분석에는 비효율적이다.

또한 기존 시스템은 보통 단일 모델 기반으로 풀이를 제공한다. 다양한 풀이 전략이 존재함에도 불구하고, 모델 내부의 사고 과정은 비교되거나 분석되지 않는다. 즉, 사용자는 하나의 풀이 결과만을 제공받으며, **여러 모델의 풀이 경향성이나 전략적 차이**를 확인할 수 없다.

이에 본 연구는 페이지 단위로 주어진 수학 문제지를 자동으로 디지털화하고, 문항 단위로 구조화한 뒤, 다양한 LLM/VLM 조합을 병렬 실행하여 풀이 과정을 비교·분석할 수 있는 End-to-End 시스템을 제안한다.

**RQ1: 페이지 단위 수학 문제지에서 문항을 정확히 자동 추출할 수 있는가?**
**RQ2: 다양한 LLM/VLM은 동일한 문제에 대해 서로 다른 풀이 전략을 생성하는가?**
**RQ3: 모델들의 풀이 과정을 구조적으로 정렬하면 모델별 경향성을 분석할 수 있는가?**

본 연구는

(1) 페이지 단위 입력을 받아 문항을 자동 분해하고 구조화하며,

(2) 각 문항에 대해 다양한 모델 조합을 병렬 실행하여 풀이를 생성하고,

(3) 풀이 과정을 구조적으로 추상화하여 전략을 비교하고 경향성을 분석한다.

이를 통해 단일 정답 제공을 넘어, **여러 풀이 방법을 동시에 제시하고 모델 간 사고 방식 차이를 관찰할 수 있는 수학 문제 분석 플랫폼**을 구현한다.
