# 선행연구 조사 Claude 프롬프트

## 🎯 핵심 프롬프트 (Claude Chat에 바로 사용)

```
당신은 AI/NLP 연구 조사 전문가입니다. 다음 주제에 대한 선행연구를 조사해주세요:

**연구 주제**:
여러 AI 모델이 생성한 수학 문제 풀이 과정을 단계별로 정렬(alignment)하고 의미적으로 그룹핑하는 방법론

**구체적 질문**:
1. 병렬로 생성된 수학 문제 풀이를 "생각의 단위(reasoning step)"로 alignment 하는 연구가 있는가?
2. Aligned 된 풀이 단계들을 대분류/계층적으로 묶는 연구가 있는가?
3. Mathematical reasoning의 step-level decomposition/annotation 연구는?
4. 여러 solution path를 비교하고 정렬하는 연구는?

**검색 키워드**:
- "mathematical reasoning step alignment"
- "solution trace comparison"
- "parallel solution annotation"
- "reasoning step grouping"
- "math solution decomposition"
- "educational concept mapping"

**원하는 정보 (각 논문마다)**:
- 제목, 저자, 학회/년도
- 핵심 방법론 (어떻게 alignment/grouping?)
- 사용 데이터셋
- 평가 메트릭
- 우리 연구와의 관련도 (1-5점)

**검색 범위**:
- 최근 5년 (2020-2025)
- NeurIPS, ICML, ICLR, ACL, EMNLP, AIED 중심

**출력 형식**:
### [논문 제목] (년도)
- **저자/학회**:
- **방법론**:
- **데이터**:
- **메트릭**:
- **관련도**: ★★★★☆
- **URL**:

마지막에 요약:
- 가장 관련 깊은 연구 Top 3
- 우리 문제와의 차이점 (gap)
- 아직 해결 안 된 문제
```

## 📊 옵션 선택 가이드

### 조사 깊이 선택

**빠른 조사 (1-2시간)**
```
위 프롬프트에 추가:

"최근 2년만 보고, 각 논문은 초록만 읽고 5줄 요약으로"
```

**표준 조사 (3-5시간)**
```
"최근 5년, 각 논문 Introduction + Conclusion 읽고 상세 분석"
```

**심층 조사 (1-2일)**
```
"최근 10년, 인용 추적 포함, survey 논문 수준으로 작성"
```

### 출력 형식 선택

**리스트 형식**
```
"논문 리스트를 표로 만들어줘 (제목 | 방법 | 데이터 | 관련도)"
```

**비교 분석**
```
"각 방법론의 장단점을 비교표로 만들어줘"
```

**시각화**
```
"연구들의 시간별 발전 흐름을 설명해줘"
```

## 🔄 단계별 심화 (선택사항)

### Step 1: 초기 조사
위의 핵심 프롬프트 실행 → 결과 확인

### Step 2: 방법론 심화
```
"위에서 찾은 논문 중 [논문명]의 방법론을 더 상세히 분석해줘:
- 구체적인 알고리즘
- 입력/출력 형식
- 우리 데이터에 적용 가능성"
```

### Step 3: 데이터셋 조사
```
"수학 문제 풀이와 관련된 공개 데이터셋을 조사해줘:
- 데이터셋 이름, 크기
- Annotation 종류 (step-level?)
- Multiple solution 포함 여부
- 라이센스"
```

### Step 4: 메트릭 조사
```
"위 연구들에서 사용한 평가 메트릭을 정리해줘:
- Alignment 품질 측정 방법
- Grouping 품질 측정 방법
- 사람 평가 프로토콜"
```

### Step 5: Gap 분석
```
"위 연구들과 우리 문제의 차이점을 분석해줘:

**우리 상황**:
- 3개 AI 모델이 같은 문제를 푼 결과
- 각 모델의 풀이 과정을 step으로 나눔
- 의미적으로 유사한 step들을 그룹핑
- 한국 수능 데이터 (영어 논문은 대부분 영어 데이터)

**차이점 분석**:
- 기존 연구로 해결 가능한 부분은?
- 우리만의 독특한 챌린지는?
- 새로운 접근이 필요한 부분은?"
```

## 💡 활용 팁

### Claude Projects 활용
```
1. Claude Projects에 새 프로젝트 생성: "Flow Map Literature Review"
2. 첫 프롬프트 실행 후 결과를 프로젝트에 저장
3. 이후 심화 질문들을 같은 프로젝트에서 계속 진행
   → 컨텍스트가 유지되어 일관된 조사 가능
```

### 검색 키워드 확장
```
"위에서 찾은 논문들이 자주 인용하는 키워드나 용어를 추가로 알려줘"
```

### 최신 연구 추적
```
"2024-2025년 발표된 최신 연구 중 주목할 만한 것은?"
```

### 관련 연구자 찾기
```
"이 분야에서 활발히 연구하는 주요 연구자/그룹은?"
```

## 📝 선행연구가 없을 경우

```
**상황**: 조사 결과, 정확히 일치하는 연구가 없음

**다음 질문**:
1. 이 문제를 새로운 연구 주제로 정의하려면?
   - 왜 필요한가?
   - 해결되면 어떤 가치가?
   - 누가 사용할 것인가?

2. 정량적 평가 방법은?
   - 자동 메트릭 가능한가?
   - 사람 평가 필요한가?
   - Ground truth를 어떻게 만들 것인가?

3. 관련 분야에서 응용할 수 있는 방법론은?
   - Sequence alignment (생물정보학)?
   - Graph matching?
   - Clustering?
```

## ✅ 실행 체크리스트

```
□ Claude Chat 앱 열기
□ 핵심 프롬프트 복사 & 붙여넣기
□ 결과 확인 (10-30개 논문 리스트업)
□ 관련도 높은 논문 3-5개 선정
□ 선정된 논문들 심화 분석 (Step 2-4)
□ Gap 분석 (Step 5)
□ 최종 정리 문서 작성
```

---

## 🚀 예상 소요 시간

- **빠른 조사**: 1-2시간 → 10-20개 논문 리스트
- **표준 조사**: 3-5시간 → 30-50개 논문, 상세 분석
- **심층 조사**: 1-2일 → Survey 논문 수준

**추천**: 표준 조사로 시작 → 필요시 심화
